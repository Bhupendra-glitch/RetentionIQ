# RetentionIQ  
### SQL-Driven Customer Retention Intelligence Platform

RetentionIQ is an end-to-end customer analytics project designed to decode customer behavior, identify loyalty patterns, detect promotional dependency, and generate actionable retention strategies using **Python, SQL, and Power BI**.

---

## Project Overview

Customer retention is one of the most critical business growth factors for modern digital brands.

This project analyzes transactional customer data to:

- Identify high-value loyal customers
- Detect discount-dependent customers
- Analyze purchasing behavior
- Segment customers based on retention probability
- Generate data-driven business recommendations
- Build interactive dashboards for strategic decision-making

---

## Problem Statement

Businesses often struggle to answer:

- Which customers are truly loyal?
- Are customers buying because of value or discounts?
- Which customers are at risk of churn?
- How can retention be improved without increasing promotional dependency?

RetentionIQ solves these challenges using data-driven analytics.
